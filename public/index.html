Tentang Sistem V1

Jalur keputusan utama tetap dari classifier doc-level IndoBERT. Analisis kalimat dan topik ditambahkan sebagai post-processing inferensi untuk membantu interpretasi hasil.

  • Model utama: indolem/indobert-base-uncased
  • Output: label dokumen + kalimat + topic badge per paragraf
  • Deploy: FastAPI di Hugging Face Spaces, frontend di Vercel

Model dilatih di Google Colab dan disajikan via FastAPI di Hugging Face Spaces.
